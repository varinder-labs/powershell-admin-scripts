# PowerShell Admin Scripts

PowerShell scripts for Windows administration, endpoint troubleshooting, and networking diagnostics.

## Scripts

### Get-SystemInfo.ps1
Displays:
- Computer name
- Logged in user
- Windows version
- IP addresses
- Disk usage

- Windows administration
- Network diagnostics
